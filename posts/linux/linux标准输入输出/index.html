Linux 标准输入输出

2021年2月12日
| Linux

  1. 标准输出输出设备

    0代表标准输入

    1代表标准输出

    2代表标错误输出

  2. 空设备文件

    /dev/null代表Linux的空设备文件

  3. 重定向到”黑洞“

    1>/dev/null 代表标准输出重定向到”黑洞“；

    2>/dev/null代表错误输出重定向到”黑洞“；

  4. 重定向输入输出

    &1表示标准输出设备；

    2>&1代表错误输出重定向到标准输出；

    2>1则会把错误输出到文件1，&1才表明是标准输出；
