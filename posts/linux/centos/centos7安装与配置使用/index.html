Centos7 安装

2021年2月11日
| Linux Centos7

一、下载

镜像下载地址

1
2

http://mirrors.aliyun.com/centos/7.7.1908/isos/x86_64/CentOS-7-x86_64-DVD-1908.iso

二、安装

1.分区（新手方案）

​ 启动分区/boot ：512M或1024M

swap分区 ：内存的2倍

​ 根分区 / ：剩余的空间

2.设置静态IP及DNS

设置静态IP

1
2
3
4
5
6
7
8

vi /etc/sysconfig/network-scripts/ifcfg-ens33
修改为：
	BOOTPROTO=static      # 静态IP
	ONBOOT=yes			  # 开机启动
	
	IPADDR=192.168.80.130 # IP
	NETMASK=255.255.255.0 # 子网掩码
	GATEWAY=192.168.80.2  # 网关

添加DNS

1
2
3
4
5
6
7
8
9

vi /etc/resolv.conf
修改为：
	nameserver 8.8.8.8
	nameserver 114.114.114.114
或
	vi /etc/sysconfig/network-scripts/ifcfg-ens33
修改为：
	DNS1=8.8.8.8
	DNS2=114.114.114.114

更新缓存:

1
2

四、安装C/C++编译器

1

yum install gcc gcc-c++

开发环境安装 参考

1

yum -y groupinstall "Development tools"

五、自动更新系统时间

设置时区

1
2

rm -f /etc/localtime
ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

安装ntp并配置

1
2
3
4

yum install ntp -y		# 安装 ntp
systemctl start ntpd	# 启动 ntp
systemctl enable ntpd	# 开机启动 ntp
service ntpd status

修改配置文件

vi /etc/ntp.conf

1
2
3

server 0.pool.ntp.org
server 1.pool.ntp.org
server 2.pool.ntp.org

六、防火墙设置

参考

列出开放的端口

1
2

firewall-cmd --list-ports		# 列出所有开放端口
firewall-cmd --list-all			# 列出详细信息

添加80、8080端口到防火墙，即放开80、8080端口。

1
2
3

firewall-cmd --zone=public --permanent --add-port=80/tcp 
firewall-cmd --zone=public --permanent --add-port=8080/tcp
firewall-cmd --zone=public --permanent --add-port=1314/tcp

重启firewalld服务，生效配置

1

systemctl restart firewalld

七、脚本加入开机启动

1
2

# 脚本加入开机启动
echo "/bin/sh /server/scripts/inotify.sh &" >> /etc/rc.local		# /etc/rc.local脚本在开机时被执行
